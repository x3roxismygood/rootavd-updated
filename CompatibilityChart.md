### Notes
* **API 28 (Pie) is not supported** — see the [system-as-root partitioning docs](https://source.android.com/devices/bootloader/partitions/system-as-root#sar-partitioning) for why.
* **Android 14+** (API 34+) ships `init_boot.img` separate from `boot.img`. `rootAVD` patches `ramdisk.img` directly so this is transparent, but if you patch a `*.img` via the `FAKEBOOTIMG` flow, point it at the right image.
* **Android 15** (API 35) introduces 16k-page-size support — Magisk **v28.0+** is required.
* **Android 16** (API 36) — including the **QPR2** sepolicy binary format — needs Magisk **v30.7+**.
* **64-Bit-only system images** (no 32-bit binaries in `lib/`) require Magisk **v23.x+**.
* The download links above point to the **official Google** system image archives; you can also install them via Android Studio → SDK Manager.
